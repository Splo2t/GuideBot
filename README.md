# SAP_GuideBot
광주서석고등학교 자율동아리 SAP 의 활동주제였던 안내로봇 소스입니다. 십자바퀴 구조로 계단 오르기가 가능하고 직접 개발한 특징점 찾기 알고리즘을 이용해 방향을 찾아갑니다.
